1. Backend:
   cd backend/back
   pip install -r requirements.txt
   python manage.py migrate

   npm install

================================================================================
                          ⚠️ PROBLEMAS CONOCIDOS
================================================================================

1. ADMIN DE DJANGO NO FUNCIONA
   - Causa: Python 3.14 incompatible con Django admin
   - Impacto: No poder acceder a /admin
   - Solución: 
     a) Usar API REST Browsable en http://localhost:8000/api/
     b) Gestionar usuarios desde frontend
     c) Downgrade a Python 3.11 o 3.12

2. ERRORES DE HIDRATACIÓN (React)
   - Causa: Extensiones del navegador modifican HTML
   - Impacto: Advertencias en consola (no afecta funcionamiento)
   - Solución: Usar suppressHydrationWarning en componentes dinámicos

3. PÁGINAS SECUNDARIAS CON MOCKS
   - Archivos: 
     - components/procesos/proceso-detalle-view.tsx
     - app/trace/page.tsx
     - app/verify/[code]/page.tsx
   - Impacto: Bajo (páginas secundarias)
   - Prioridad: Media (pueden actualizarse después)
